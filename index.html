myFlix

Overview

Once opening this app, you can register an account or login to an existing one. Once logged in the program with route you to the movie screen where you can see a list of movies. From here you have the freedom to see information from any of the following movies, edit your account, or logout.

Objective

Using Angular, build the client-side for an application called myFlix based on its existing server-side code (REST API and database), with supporting documentation.

Tools

This project was started by building an api by using a REST architecture, getting all of our endpoints set up, and inputting all our information into MongoDB. It was written using the React library, styled using bootstrap, and hosted online through gh-pages.
